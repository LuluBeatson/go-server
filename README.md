# Go Server

This project combines a series of projects from this [tutorial](https://www.youtube.com/watch?v=jFfo23yIWac) from freeCodeCamp.org.

# Start

```bash
go build
go run ./
```
Starts a server on port 8080. To use a different port:

```bash
go run ./ -port=5000
```
